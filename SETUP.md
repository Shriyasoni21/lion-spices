## 🚀 Quick Start

### Step 1: Install Dependencies
```bash
cd spices
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

The website will automatically open at `http://localhost:3000`

### Step 3: Build for Production
```bash
npm run build
```

Production files will be in the `dist/` folder.

## 🔧 Customization Guide

### Change Colors
Edit `tailwind.config.js` in the `colors` section to match your brand.

### Update Products
Edit `src/data/productData.js` to add/modify products, categories, and features.

### Update Blogs & Testimonials
Edit `src/data/blogData.js` to update blog posts and customer testimonials.

### Modify Sections
All section components are in `src/components/sections/` - fully customizable.
